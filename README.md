# todo-list

| HTTP Method | Action                            | Examples                 |
|-------------|-----------------------------------|--------------------------|
| GET         | Obtain information about all task | /get                     |
| POST        | Create new task                   | /add/{description}       |
| PUT         | Update status of task             | /mark/{todo_id}/{status} |
| DELETE      | Delete task                       | /remove/{todo_id}        |
